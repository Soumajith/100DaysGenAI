# FAQ - Frequently Asked Questions

## General Questions

### What is this 100 Days of GenAI challenge?
A structured learning path to go from beginner to proficient in Generative AI over 100 consecutive days.

### Do I need prior experience?
Basic programming knowledge helps, but complete beginners can start with extra time on fundamentals.

### How much time per day?
Minimum 2 hours, recommended 3-4 hours. Adjust based on your schedule and learning pace.

### What if I miss a day?
Try not to! Consistency is key. If you miss a day, continue from where you left off. Don't restart.

### Can I go faster or slower?
Yes! This is a guideline. Adjust pace based on your understanding and time availability.

## Technical Questions

### Which framework should I use - PyTorch or TensorFlow?
**PyTorch** is recommended for learning (more intuitive, research-focused). TensorFlow is also good (production-ready).

### Do I need a GPU?
Not initially. Use Google Colab (free GPU) for intensive tasks. For serious work, consider cloud GPUs or local GPU.

### What if I don't understand something?
1. Re-read the material
2. Watch video tutorials
3. Ask in forums (Reddit, Stack Overflow)
4. Experiment with code
5. Reach out to the community

### Should I read research papers?
Yes! Start with seminal papers (Attention Is All You Need, GPT, BERT). You won't understand everything at first - that's okay.

### How do I stay updated?
- Follow AI newsletters (The Batch, Import AI)
- Twitter/X: Follow AI researchers
- Reddit: r/MachineLearning
- Papers with Code
- Hugging Face blog

## Troubleshooting

### Code isn't working
1. Read error messages carefully
2. Check documentation
3. Google the error
4. Simplify to minimal example
5. Ask for help with code snippet

### Can't install libraries
```bash
# Try upgrading pip
pip install --upgrade pip

# Use virtual environment
python -m venv env
source env/bin/activate  # or env\Scripts\activate on Windows

# Install specific versions
pip install torch==2.0.0
```

### Out of memory errors
- Reduce batch size
- Use gradient accumulation
- Use mixed precision (fp16)
- Clear GPU cache
- Use smaller models

## Motivation

### I'm falling behind, should I quit?
**No!** It's not a race. Even if you take 150 days, you'll have learned more than if you quit.

### Feeling overwhelmed?
Normal! AI is vast. Focus on one concept at a time. Break big topics into small chunks.

### Not seeing progress?
You are progressing! Keep a learning journal to see how far you've come. Compare to yourself, not others.

### How to stay motivated?
- Set small, achievable goals
- Celebrate small wins
- Join community
- Build projects you care about
- Remember why you started
