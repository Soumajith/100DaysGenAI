# 🚀 100 Days of Generative AI

Welcome to your 100-day journey to master Generative AI! This repository is designed to take you from beginner to proficient in GenAI through structured, daily learning.

## 📚 Overview

This learning path covers:
- **Fundamentals**: AI/ML basics, Python programming, and mathematics
- **Deep Learning**: Neural networks, transformers, and architectures
- **Generative Models**: GANs, VAEs, Diffusion Models
- **Large Language Models**: GPT, BERT, and transformer architectures
- **Practical Applications**: Building real-world GenAI applications
- **Advanced Topics**: Fine-tuning, RAG, agents, and deployment

## 🎯 Learning Structure

The 100 days are organized into 14 weeks + final projects:

### **Weeks 1-2: Foundations (Days 1-14)**
- Python for AI/ML
- Linear Algebra, Calculus, Probability
- Introduction to Machine Learning
- Neural Networks Basics

### **Weeks 3-4: Deep Learning Fundamentals (Days 15-28)**
- Deep Neural Networks
- CNNs and Computer Vision
- RNNs and Sequential Data
- PyTorch/TensorFlow basics

### **Weeks 5-6: Transformer Architecture (Days 29-42)**
- Attention Mechanisms
- Transformer Architecture Deep Dive
- BERT and Encoder Models
- GPT and Decoder Models

### **Weeks 7-8: Generative Models (Days 43-56)**
- Generative Adversarial Networks (GANs)
- Variational Autoencoders (VAEs)
- Diffusion Models (Stable Diffusion, DALL-E)
- Image Generation Techniques

### **Weeks 9-10: Large Language Models (Days 57-70)**
- LLM Fundamentals
- Prompt Engineering
- OpenAI API, Anthropic Claude, Google Gemini
- Model Training and Fine-tuning

### **Weeks 11-12: Advanced LLM Techniques (Days 71-84)**
- Retrieval-Augmented Generation (RAG)
- Vector Databases (Pinecone, Weaviate, ChromaDB)
- LangChain and LlamaIndex
- AI Agents and Multi-Agent Systems

### **Week 13: Production & Deployment (Days 85-91)**
- Model Optimization and Quantization
- API Development (FastAPI, Flask)
- Cloud Deployment (AWS, Azure, GCP)
- MLOps Best Practices

### **Week 14: Ethics & Advanced Topics (Days 92-98)**
- AI Safety and Alignment
- Bias and Fairness
- Responsible AI Development
- Latest Research and Trends

### **Days 99-100: Capstone Project**
- Build and deploy a complete GenAI application

## 📋 Prerequisites

- Basic programming knowledge (preferably Python)
- Understanding of basic mathematics
- A computer with at least 8GB RAM
- Curiosity and dedication!

## 📖 How to Use This Repository

1. **Fork this repository** to track your own progress
2. **Follow the daily schedule** in the `/days` directory
3. **Complete exercises and projects** for hands-on practice
4. **Document your learning** in daily logs
5. **Share your progress** on social media with #100DaysOfGenAI

## 🎓 Daily Structure

Each day includes:
- **Theory** (30-45 min): Core concepts and reading
- **Practice** (45-60 min): Coding exercises and tutorials
- **Project Work** (30-45 min): Building real applications
- **Reflection** (10-15 min): Document learnings

## 🏆 Milestones

- **Day 14**: Build your first neural network
- **Day 28**: Create a simple image classifier
- **Day 42**: Implement a transformer from scratch
- **Day 56**: Generate images with a diffusion model
- **Day 70**: Fine-tune a language model
- **Day 84**: Build a RAG application
- **Day 91**: Deploy an AI application to production
- **Day 100**: Complete capstone project
