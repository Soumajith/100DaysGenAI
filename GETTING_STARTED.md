# Getting Started with 100 Days of GenAI

## 🎯 How to Begin

Welcome! Here's your step-by-step guide to starting your GenAI learning journey.

## Step 1: Set Up Your Environment

### Install Python
```bash
# Check if Python is installed
python --version  # Should be 3.8 or higher

# If not installed, download from python.org
```

### Create a Virtual Environment
```bash
# Create a new environment
python -m venv genai-env

# Activate it
# On Windows:
genai-env\Scripts\activate
# On Mac/Linux:
source genai-env/bin/activate
```

### Install Essential Libraries
```bash
pip install numpy pandas matplotlib jupyter
pip install torch torchvision  # PyTorch
pip install scikit-learn
```

## Step 2: Set Up Your Workspace

1. **Fork this repository** to your GitHub account
2. **Clone it** to your local machine
   ```bash
   git clone https://github.com/YOUR-USERNAME/100DaysGenAI.git
   cd 100DaysGenAI
   ```
3. **Create a branch** for your learning journey
   ```bash
   git checkout -b my-100days-journey
   ```

## Step 3: Plan Your Schedule

### Daily Time Commitment
- **Minimum:** 2 hours per day
- **Recommended:** 3-4 hours per day
- **Ideal time:** Morning or evening when you're most focused

### Weekly Schedule Example
- **Weekdays:** 2-3 hours (theory + practice)
- **Weekends:** 4-5 hours (projects + review)

### Best Practices
- ✅ Study at the same time each day
- ✅ Take breaks every 45-60 minutes
- ✅ Practice coding daily
- ✅ Review previous concepts weekly
- ❌ Don't skip days (consistency is key!)

## Step 4: Track Your Progress

- Resources used

## 📋 Learning Tips

### For Beginners
1. **Don't rush** - Understanding fundamentals is crucial
2. **Code along** - Type every code example yourself
3. **Ask questions** - Use Stack Overflow, Reddit, Discord
4. **Build projects** - Apply concepts immediately
5. **Review regularly** - Revisit difficult topics

### For Intermediate Learners
1. **Deep dive into papers** - Read original research
2. **Implement from scratch** - Build models without libraries
3. **Contribute to open source** - Join AI projects
4. **Experiment freely** - Try different architectures
5. **Teach others** - Write tutorials, create content

### For Everyone
- 🔥 **Stay consistent** - 100 days straight!
- 💪 **Push through plateaus** - Growth isn't linear
- 🤝 **Join the community** - Learn together
- 🎯 **Focus on understanding** - Not just completion
- 🚀 **Build real projects** - Create a portfolio
